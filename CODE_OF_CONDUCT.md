# Code of conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus criticism on code and
ideas rather than people, and keep the project a welcoming place for
researchers and practitioners alike.

Violations can be reported privately via
[GitHub security advisories](https://github.com/skortmann/hybrid-vpp-rl/security/advisories/new)
or a direct message to the repository owner.
